# jQuery 基础笔记

* ```jQuery```是一个```JavaScript```脚本库，不需要特别的安装，只需要我们在页面 ```<head>``` 标签内中，通过 ```script``` 标签引入 ```jQuery``` 库即可
    * 比如： 

    ```
    <script type="text/javascript" src="http://libs.baidu.com/jquery/1.9.1/jquery.js"></script>

    ```
